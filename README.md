# Таблица умножения (C#)

Консольная программа, выводящая таблицу умножения для числа от 1 до 10.

## Как работает
1. Запрашивает число от 1 до 10
2. Проверяет корректность ввода
3. Выводит 10 строк таблицы умножения

## Пример
```
Введите число от 1 до 10: 5

5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
...
5 x 10 = 50
```

## Запуск
Скомпилируйте в Visual Studio и запустите (Ctrl+F5).
